Ubah gambar apa pun menjadi prompt AI

Upload foto, lalu dapatkan prompt siap pakai untuk Midjourney, Stable Diffusion, DALL·E, atau Flux.

Hasil Prompt

Tips:

  • Gunakan gambar dengan resolusi jelas untuk hasil terbaik.
  • Pilih gaya prompt sesuai tool AI yang kamu gunakan.
  • Prompt bisa kamu edit lebih lanjut sesuai kebutuhan.

Pengaturan API Key

Aplikasi ini menggunakan Google Gemini API (gratis). API key disimpan hanya di browser kamu (localStorage), tidak dikirim ke server mana pun.

Belum punya? Dapatkan gratis di Google AI Studio.
