🎨 Image to Prompt

Ubah gambar apa pun menjadi prompt AI

Upload foto, lalu dapatkan prompt siap pakai untuk Midjourney, Stable Diffusion, DALL·E, Flux, atau Danbooru.

Hasil Prompt

💡 Tips

  • Gunakan gambar dengan resolusi jelas untuk hasil terbaik.
  • Pilih gaya prompt sesuai tool AI yang kamu gunakan.
  • Prompt bisa kamu edit lebih lanjut sesuai kebutuhan.
  • Tekan Ctrl + V untuk paste gambar langsung dari clipboard.

Pengaturan API Key

Aplikasi ini menggunakan Google Gemini API (tersedia gratis). API key disimpan hanya di browser kamu (localStorage) — tidak pernah dikirim ke server mana pun.

Belum punya? Dapatkan gratis di Google AI Studio.
